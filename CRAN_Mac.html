R Download Instructions for Mac OS

Curry W. Hilton

July 5, 2016

  • Follow link below to CRAN

CRAN

  • Choose “Download R for (Mac) OS X”
  • If,
    • Mac OS X is version 10.9 (Maverick) or higher, choose R-3.3.1.pkg
    • Mac OS X is version 10.6 (Snow Leopard) - 10.8 (Mountain Lion), choose R-3.2.1-snowleopard.pkg
  • Once downloaded, choose “run” to install
